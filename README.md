# Indoor-Farming-Intelligence-Suite
Indoor Farming Intelligence Suite is an AI-powered web application designed to modernize indoor agriculture through a unified ecosystem for aeroponic and hydroponic farming. The platform offers intelligent pre-setup consultation to guide farmers on setup costs, space requirements, crop selection, and profitability analysis. It enables continuous environmental management by monitoring key parameters such as light intensity, humidity, and moisture and providing AI-driven recommendations. An integrated vendor marketplace connects farmers with verified vendors for purchasing seeds and selling harvested crops. With features like OTP-based registration, Razorpay subscription management, invoice generation via email, and a chatbot for advisory support. 
The system aims to reduce entry barriers for new farmers, increase crop yield, and deliver a user-friendly, all-in-one digital solution for indoor farming management. Using technologies like Spring Boot, React JS and MySQL for database.
